# ReBite

ReBite is a web platform that connects restaurants with surplus food to volunteers and NGOs, helping reduce food waste while delivering meals to people who need them.

## Problem

A large amount of edible food is wasted every day by restaurants, events, and food businesses, while many communities still struggle with food insecurity. The main problem is not always lack of food, but lack of connection between food donors and people who can distribute it.

## Solution

ReBite creates a simple bridge between three groups:

* Restaurants post available surplus food.
* Volunteers claim deliveries.
* NGOs receive and distribute meals.

## Key Features

* Food donation posting form
* Live donation cards
* Volunteer claim system
* Delivery status tracking
* NGO dashboard
* Impact counters for meals saved, active deliveries, and NGOs helped
* Clean and responsive web interface

## Tech Stack

* HTML
* CSS
* JavaScript

## Impact

ReBite helps turn potential food waste into social good by making food rescue faster, easier, and more organized.
